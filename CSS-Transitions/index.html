Transitions

If I use the term "all", what can I delete from the following code?

transition: background .3s, ease color 0.2s linear;

If a designer wants to use transitions with the same effects then you would replace both "background .3s" and color, with "all". This allows to cover all the possible properties that can be transitioned.

Define the following timing functions/transitioning elements:

"CSS Transitions allow property changes in CSS values to occur smoothly over a specified duration."

  1. Ease: Transiton that goes from slow, to fast, then slow.
  2. Linear: Transition with a consistent speed.
  3. Ease-In: A slow start transition.
  4. ease-Out:Transtion that both begins and ends slow
  5. Cubic-Bezzier: transition with custom values defined in the Cubic Bezzier function.
